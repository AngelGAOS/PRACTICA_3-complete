##  About The Project
This repository contains the software extension developed for **Practice 3** of the Theory of Computation course at ESCOM. The project extends a DFA simulator to support non-determinism, state closures, and optimization algorithms.

### Authors
* **Angel Heron Garcia Osornio**
* **Alan Gabriel Ramírez Nolasco**

---

##  Core Features
* **NFA Simulation:** Supports multiple transitions per symbol.
* **NFA-λ Simulation:** Implements lambda-closure calculations.
* **Subset Construction:** Converts NFA/NFA-λ to DFA.
* **Hopcroft's Minimization:** Reduces DFA to its minimal form.
* **Batch Testing:** Mass string validation via .txt files.
